### Sub `incr()`
Replaces code like i=i+1, or i=i+value
can be used not only for Numbers


For example:
<code>

 i=1
 incr i ' i = 2

 n=0.5
 incr n, 0.3 ' n = 0.8

 s="Hello, "
 incr s, "World!" ' s = "Hello, World!"

</code>


parameters: 
</summary>
